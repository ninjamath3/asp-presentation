# Alien Surveillance Protocol (ASP)

**Genre:** Augmented Reality Geolocation Game  
**Platforms:** iOS & Android  
**Built with:** React Native & Expo  

---

## Overview

Alien Surveillance Protocol (ASP) is a mobile AR game where players defend their sector against unpredictable alien invasions. 
By exploring real-world locations, scanning for hidden aliens, and managing strategic combat through a console interface, players are immersed in a unique augmented reality experience.

This repository is intended to showcase the game’s concept, interface, and key gameplay elements. Source codes are maintained privately.

---

## Key Features

- **Geolocation-based gameplay:** Aliens appear according to the player’s location.  
- **AR scanning:** Players use their device to detect and track aliens in augmented reality.  
- **Strategic combat:** Engage different alien types using the base’s defenses and weapons.  
- **Base management:** Upgrade and maintain your base to withstand ongoing invasions.  
- **Dynamic alien behaviors:** Each alien type presents unique challenges and requires specific strategies.
